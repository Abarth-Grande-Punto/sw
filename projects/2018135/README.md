# Παραδοτέο 4

Για το τέταρτο παραδοτέο πρέπει να γίνει χρήση ενός λειτουργικού χωρίς systemd, έτσι αποφάσισα να εγκαταστήσω το plan9 σε VirtualBox. Μου φάνηκε ενδιαφέρον και θα ήθελα να δουλέψω σε αυτό καθώς πιστέυω οτί θα πάρω σημαντικές γνώσεις που θα μου χρειαστούν και σε άλλα νεότερα λειτουργικά, καθώς αν και το είναι plan9 αρκετά παλιό αποτελεί μια πολύ ενδιαφέρον βάση για μελέτη. Έκανα κάποιες δοκιμές, σύνδεση στο διαδίκτυο και πειραματισμό ώστε να εξοικειωθώ με το λειτουργικο. Κάποια screenshots του plan9:
