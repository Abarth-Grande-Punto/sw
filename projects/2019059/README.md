## <a name="P"> ΕΙΣΑΓΩΓΗ:</a>
Οι στόχοι μου για το μάθημα «Τεχνολογία Λογισμικού» είναι να διευρύνω τις γνώσεις που απέκτησα από το μάθημα «Επικοινωνία Ανθρώπου και Υπολογιστή», δηλαδή την χρήση του terminal των linux . Εκτός από αυτό σκοπεύω να μάθω το εργαλείο docker το οποίο θα με βοηθήσει πολύ στην μελλοντική μου καριέρα ως προγραμματιστής. Επιπλέον, μέσα από τις ασκήσεις βιογραφικού μαθαίνουμε να φτιάχνουμε βιογραφικά τα οποία μπορούμε να γνωρίσουμε τις ικανότητες μας και να τα χρησιμοποιήσουμε για την δημιουργία δικού μας βιογραφικού. Μέσα από της ασκήσεις γραμμής εντολών επιθυμώ να μάθω νέες λειτουργίες της γραμμής εντολών που με ενδιαφέρουν και θα με βοηθήσουν στη μελλοντική μου καριέρα. Τέλος, οφείλω να μάθω να συνεργάζομαι σαν ενεργό μέλος της ομάδας ομάδας μου μέσω του GitHub , κάτι το οποίο θα με βοηθήσει καθημερινά στο εργασιακό μου περιβάλλον (χρήση GitHub chat και pull request).

## update
Έχοντας ολοκληρώσει την μισή χρόνια πιστεύω πως έχω εξηκοιωθεί με τις απαιτήσεις του μαθήματος. Σχετικά με τις γνώσεις που ανφερόμουν πιστεύω πως έχω αποκτήσει ένα μεγάλο ποσοστό. Ιδιαίτερα μου κίνησαν το ενδιαφέρον δυο θέματα που είχε αναφερθεί ο Alan Kay. Το πρώτο θέμα σχετιζόταν με την αρχιτεκτονική συστημάτων και ανέφερε δυο κατηγορίες συστημάτων. Ως πυραμίδα ανέφερε την πρώτη κατηγορία, τα συστήματα αυτά δεν έχουν πραγματική αρχιτεκτονική και για αυτό δημιουργούνται πολλά προβλήματα. Αυτό συμβαίνει διότι οι δημιουργοί τους απλά "αναβαθμίζουν" προιγούμενα συστήματα χωρίς κάποιο σχέδιο και οργάνωση, με αποτέλεσμα το "καινούριο" σύστημα να είναι το στην πραγματικότητα το αρχικό με "καλύτερες" λειτουργίες. Αντίθετα, τα λειτουργικά Empire State Building χαρακτηρίζονται απο πολύ οργανομένη δομή και αποτέλεσμα της ομαδικότητας ατόμων με όραμα. Επιπλέον, η ιδέολογία του Alan Key σχετικά με την ανάγκη ενός οράματος μου τράβηξε ιδιαίτερα το ενδιαφέρον. Σύμφωνα με αυτήν δεν αρκεί να έχεις γνώσεις και να ξέρεις να προγραμματίζεις αλλά απαιτήται η "οπτική" στο μέλον προκειμένου το άτομο να καινοτομήσει, αλλιώς απλά θα αντιγράφει τους παλαιώτερους. Έχοντας περάσει το μάθημα θα ήθελα ,εκτός από γνώσεις , να αποκτήσω όραμα και τρόπο σκέψης ώστε να μπορέσω να συνησφέρω στον τομέα που θέλω να ακολουθήσω.
