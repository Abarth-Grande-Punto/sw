# ΠΑΡΑΔΟΤΕΟ 1

## ΕΙΣΑΓΩΓΗ
Ένας απο τους στόχους που έχω σχετικά με το μάθημα της «Τεχνολογίας Λογισμικού» είναι η καλύτερη κατανόηση της έννοιας του λογισμικού και να δοκιμάσω διαφορετικά λογισμικά (ArchLinux, Ubuntu κτλ). Γενικότερα, η ιδέα του open source λογισμικού μου αρέσει πολύ και προσπαθώ όσο περισσότερο μπορώ να το χρησιμοποιώ. Επίσης, η συνεργασία με άλλους συμφοιτητές μου, για την ανάπτυξη λογισμικού, πιστεύω ότι θα μου προσφέρει διάφορες γνώσεις, τόσο σε κοινωνικό επίπεδο όσο και σε προγραμματιστικό, διότι δεν γνωρίζω όλες τις γλώσσες προγραμματισμού. Πιστεύω ότι το μάθημα θα μου δώσει τις απαραίτητες βάσεις πάνω στη σχεδίαση και ανάπτυξη λογισμικού και επιπλέον εμπειρία, που θα αποκτήσω απο τη χρήση διαφορετικών λογισμικών ώστε να έχω μια σφαιρική εικόνα σ'αυτά. Επίσης, πιστεύω ότι τα βίντεο τα οποία θα παρακολουθήσω, θα μου προσφέρουν χρήσιμες πληροφορίες και γνώσεις πάνω στο αντικείμενο του λογισμικού. 

Τα τελευταία χρόνια οφείλω να ομολογήσω ότι χρησιμοποιούσα σχεδόν αποκλειστικά έτοιμο λογισμικό , με εξαίρεση την ενασχόληση μου για ένα μικρό χρονικό διάστημα με τα ArchLinux. Απο το έτοιμο λογισμικό θεωρώ ότι αυτά που έμαθα και ο τρόπος με τον οποίο έκανα το οτιδήποτε σ'αυτά τα λογισμικά, ήταν απερίγραπτα περιοριστικά και προωθούσαν το πως ήθελαν οι εταιρίες να χρησιμοποιούν οι χρήστες το λογισμικό τους. Αυτό που ευελπιστώ μέσω του μαθήματος είναι να μάθω περισσότερα σχετικά με το καλής ποιότητας και ανοιχτού κώδικα λογισμικό και να καταλάβω καλύτερα τις λειτουργίες και τις διαδικασίες που τρέχουν στο παρασκήνιο όσο χρησιμοποιείται το λογισμικό.

## ΑΣΚΗΣΗ ΓΡΑΜΜΗΣ ΕΝΤΟΛΩΝ - TWILIO

Για το 11ο παραδοτέο, υλοποίησα μια απο τις ασκήσεις απο τη κατηγορία του software και συγκεκριμένα το εργαλείο [twilio-python](https://github.com/twilio/twilio-python). Μου φάνηκε ενδιαφέρον διότι χρησιμοποιεί python scripts για να πραγματοποιήσει κλήση και αποστολή μηνύματος στο κινητό τηλέφωνο.

Αρχικά, έπρεπε να δημιουργήσω λογαριασμό στο [Twilio](https://www.twilio.com/) και να συνδέσω σ'αυτόν τον αριθμό του τηλεφώνου μου. Στη συνέχεια, μου δόθηκε ένας αριθμός Twilio ο οποίος θα ήταν ο αριθμός που θα έκανε την κλήση και θα έστελνε μήνυμα στον αριθμό που δήλωσα στην αρχή. Το account_sid και auth_token μου ήταν χρήσιμα για την δημιουργία και των δύο python scripts, ένα για κλήση και ένα για μήνυμα. Η εγκατάσταση του Twilio έγινε με την εντολή `pip install twilio`, σε Alpine Linux.

Στο παρακάτω asciinema υπάρχουν τα δύο python scripts και η εκτέλεση τους: https://asciinema.org/a/CW9Lp3xwlPTslioB9ouVaQdk5

# ΠΑΡΑΔΟΤΕΟ 12

## ΕΠΙΛΟΓΟΣ

Τελειώνοντας την εργασία του εξαμήνου, πιστεύω ότι απέκτησα πολλές και ενδιαφέρουσες γνώσεις πάνω στο αντικείμενο του λογισμικού. Αρχικά, τα βίντεο κουίζ μου έδωσαν μια διαφορετική οπτική γωνία για τον κόσμο των λογισμικών. Ο απεριόριστος αριθμός Linux distributions που υπάρχουν χρησιμοποιείται απο ένα πολύ μικρό πλήθος χρηστών σε συγκρισή βέβαια με τα πιο διάσημα και εμπορικά λογισμικά, όπως είναι τα Windows και τα MacOS. Γεγονός αξιοσημειώτο, αφού λογισμικά όπως το plan9 που είναι σαφώς καλυτέρο και πιο μοντέρνο, δεν είχαν την ίδια τύχη με τα Windows και τα MacOS λόγω της έλλειψης μάρκετινγκ και της συνήθειας των χρηστών στα ήδη υπάρχοντα λειτουργικά συστήματα. Επίσης, μέσω της εγκατάστασης νέου λειτουργικού συστήματος χωρίς systemd, Alpine Linux, κάταλαβα καλύτερα πως λειτουργεί και τελικά πως να το χρησιμοποιειώ σωστά για τις δικές μου ανάγκες. Όσο για τις ομάδες, κατάλαβα πόσο σημαντικό είναι το κάθε μέλος να έχει έναν ρόλο σ'αυτήν και να κατανοεί τους κανόνες που έχουν τεθεί απο την ομάδα.
