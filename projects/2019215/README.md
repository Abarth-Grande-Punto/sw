# 1ο Παραδοτέο

## Εισαγωγή
Στο μάθημα της τεχνολογίας λογισμικού φέτος θα ήθελα να επεκτείνω τις γνώσεις που είχα αποκτήσει από το hci στο github με τη χρήση των organizations καθώς είναι ένα πολύ χρήσιμο εργαλείο για το μέλλον και θα με παροτρύνει να εργαστώ συνεργατικά στην ανάπτυξη λογισμικού, κάτι που μου ακούγεται αρκετά ενδιαφέρον. Επίσης για την ανάγκη του μαθήματος θα χρησιμοποιήσω για πρώτη φορά Dockers το οποίο αποτελεί ένα πολύ σημαντικό εργαλείο. Τέλος, από την παρακολούθηση των εβδομαδιαίων βίντεο περιμένω να αποσκοπήσω ενδιαφέρον πληροφορίες.
<br><br>
Προς το παρόν μετά απο την παρακολούθηση των μισών βίντεο κατάλαβα πως υπάρχει μία καταπολέμηση των νέων ιδεών για λειτουργικά συστήματα από τις μεγάλες εταιρίες της αγοράς καθώς δεν τις συμφέρει να μαθευτούν οι ιδέες αυτές στο κόσμο ξεκάθαρα για λόγους κέρδους, ειδικά αυτών χωρίς systemd, για το λόγο αυτό αποφάσισα να ασχοληθώ και με το plan9 και το AmigaOS στο συμμετοχικό περιεχόμενο. Μου έκαναν μεγάλη εντύπωση οι καινοτομίες των λειτουργικών αυτών και η διαφορετική λειτουργία τους σε σχέση με αυτά που έχουμε συνηθίσει και πως αμα δεν ψαχτεί κάποιος αρκετά δεν θα μάθει ποτέ για αυτά.
